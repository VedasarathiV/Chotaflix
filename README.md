# ChotaFlix - Unlimited Movies & TV Shows

ChotaFlix is a premium, modern streaming platform interface designed to provide a cinematic experience for browsing movies and TV shows. Built with high-performance Vanilla CSS and JavaScript, it features a sleek dark mode, smooth animations, and a responsive layout.

![ChotaFlix Preview](index.html) <!-- Replace with an actual screenshot if available -->

## ✨ Features

- **Premium UI/UX**: Dark luxury aesthetic with vibrant gradients and glassmorphism.
- **Dynamic Hero Section**: Interactive hero banner with featured content and quick actions.
- **Responsive Content Rows**: Smooth-scrolling rows for different categories (Trending, Originals, etc.).
- **Interactive Modals**: Detailed views for each title including ratings, meta-info, and descriptions.
- **Custom Video Player**: Built-in player interface with custom controls and progress tracking.
- **Search & Filter**: Real-time search functionality to find your favorite content.
- **Mobile Optimized**: Fully responsive design that works perfectly on all devices.
